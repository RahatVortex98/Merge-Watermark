# PDF Manipulation Project

This project demonstrates how to manipulate PDF files using the `PyPDF2` library in Python. The functionalities include watermarking a PDF, counting pages, and combining multiple PDF files into one.

## Features

1. **Watermarking a PDF**: 
   - This feature allows you to apply a watermark from one PDF file to another.
  
2. **Counting Pages**: 
   - You can open a PDF file and count the number of pages it contains.
  
3. **Combining PDF Files**: 
   - Multiple PDF files can be combined into a single PDF file.

## Prerequisites

Make sure you have Python installed on your machine. You also need to install the `PyPDF2` library. You can install it using pip:
